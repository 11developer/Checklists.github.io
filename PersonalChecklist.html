Checklist

OSINT (Open Source Intelligence)

  • Subdomain Enumeration
    • amass enum -active -brute -o testoutput.txt -d test.com
    • sublist3r -d example.com -o subdomains.txt
  • Link Finder
    • (Be Sure to cd into /LinkFinder) python3 linkfinder.py -i https://test.com -o results.html
  • Metadata Extraction
    • exiftool example.jpg

Network Scanning

  • Comprehensive Nmap Scan
    • nmap -p- -sS -sU -T4 -A -v -Pn --script vuln X.X.X.X
  • Nmap Aggressive Scan with OS and Service Detection
    • nmap -A -O -sV X.X.X.X
  • Nmap Full TCP and UDP Scan with OS and Service Detection
    • nmap -sS -sU -p- -A -O X.X.X.X
  • Nmap Scan Top Ports with Vulnerability Scripts
    • nmap --top-ports 20 --script vuln X.X.X.X
  • Nmap Comprehensive Scan with Default Safe Scripts and Intense Scan Options
    • nmap -sC -A -T4 X.X.X.X
  • SSL Certificate Information
    • sslscan example.com
  • DNS Zone Transfer
    • dig axfr @ns1.example.com example.com
  • DNS Rebinding Attack
    • rebind -t example.com

Vulnerability Scanning

  • Reconnaissance with FinalRecon
    • finalrecon --full https://test.com
  • Vulnerability Scanning with OpenVAS
    • openvas -u username -w password --target example.com
  • Automated Security Testing with Nikto
    • nikto -h https://example.com
  • HTTP Smuggling Detection
    • smuggler.py -u https://example.com
  • Content Security Policy Bypass
    • csp_evaluator --url https://example.com
  • HTTP Header Security Check
    • python3 testssl.py --header https://example.com
  • Web Application Firewall Detection
    • wafw00f http://example.com
  • Security Misconfiguration Check with Lynis
    • lynis audit system -Q
  • Dependency Check for Vulnerable Libraries
    • dependency-check --project example --scan /path/to/project

Injection Testing

  • Web Application Fuzzing
    • ffuf -w /home/user5482/Documents/payloads/common.txt:FUZ -u https://test.com/FUZZ -fs 0 -mc 200
  • SQL Injection Testing
    • sqlmap -u "http://www.example.com/page.php?id=1"
  • Command Injection Testing
    • commix --url="http://example.com/index.php?id=1" --batch
  • Open Redirect Testing
    • curl -v "http://example.com/redirect?url=http://attacker.com"
  • File Inclusion Testing
    • curl "http://example.com/index.php?page=../../../../etc/passwd"

API and Web Application Testing

  • Parameter Discovery
    • arjun -u http://test.com
    • arjun -u http://test.com -o new.json
  • XSS Scanning
    • xsser -u "http://example.com/index.php?id=1" --auto
    • xsser -u "http://example.com/index.php?id=1" --auto -o results.txt
  • API Testing with Postman
    • newman run collection.json -e environment.json
  • GraphQL Endpoint Testing
    • graphqlmap -u http://example.com/graphql -H 'Authorization: Bearer '
  • Hidden Parameter Discovery
    • parameth -u https://example.com
  • HTTP Parameter Pollution
    • curl -X GET "http://example.com/index.php?param1=value1¶m1=value2"

Authentication and Session Testing

  • Brute Force Login
    • hydra -l bob -P /usr/share/wordlists/rockyou.txt -f X.X.X.X http-get /protected/
  • JWT Token Cracking
    • jwtcrack eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...

Mobile and Client-Side Security

  • Mobile App Security Testing
    • mobSF -f example.apk
  • JavaScript File Analysis
    • jsparser -u https://example.com -o results.txt
  • Cross-Origin Resource Sharing (CORS) Testing
    • python3 corsy.py -u https://example.com
  • WebSockets Security Testing
    • wstest -t https://example.com/socket

Special Case Testing

  • Subdomain Takeover
    • subjack -w subdomains.txt -t 20 -timeout 30 -ssl -c /path/to/fingerprints.json -v
  • Email Spoofing Testing
    • sendEmail -f sender@example.com -t target@example.com -u "Test" -m "This is a test email" -s smtp.example.com
  • HTTP/2 Security Testing
    • h2spec -p 443 -h example.com
